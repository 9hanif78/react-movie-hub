A modern movie discovery web application built with React + Vite that allows users to explore trending movies, search for titles, and view detailed information in a fast and responsive UI.

🚀 Features
🔍 Search movies by title
🎥 View detailed movie information (poster, rating, description)
🌟 Browse trending / popular movies
❤️ Add movies to favorites (optional feature)
📱 Fully responsive design
⚡ Fast performance with Vite
🛠️ Tech Stack
React.js
Vite
JavaScript (ES6+)
CSS / Tailwind CSS
TMDB API (The Movie Database)
